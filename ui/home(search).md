你现在的任务是：根据以下设计文档，为我生成“发现页（搜索首页）”的高保真 UI 页面效果图（Mockup）。

【视觉要求】
- 整体采用暖色调（Warm Tone），主色范围 #DDAA66 ~ #EEC08A
- 风格现代、简洁、扁平化
- 搜索框、卡片、列表项均使用统一圆角
- 图标线条统一、柔和

【页面结构】
- 顶部常驻搜索框
- 未搜索状态：
  - 播放历史（最近10首）
  - 搜索历史词（最多20条，可删除）
- 搜索后状态：
  - 搜索结果列表（封面图 + 歌名 + 歌手 + 专辑）
  - 支持分页加载更多
  - 每条结果右侧展示操作菜单：播放 / 加入队列末尾 / 下一首播放 / 加入歌单 / 收藏

【交互提示】
- 请求频繁时显示 Toast：“请求频繁，请稍后再试”

【输出要求】
- 分辨率：1440 × 1024
- 不要添加水印或额外说明
- 严格遵守设计文档，不得自由发挥
